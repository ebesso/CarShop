# Setup

1. Install packages   
```
    npm i
```
2. Set environment variables
```
API_PORT
API_SECRET
DATABASE_URL
GMAIL_PASSWORD
GMAIL_ADDRESS
````

3. Set APP Url in config.js

4. Initialize Data File

```
npm run init-data
```

5. Initialize admin user
```
npm run init-user
```
6. Start
```bash
    npm start
```
